Behavior Anchor Manifest — Containers

Every row is a user-facing capability and the single anchor test that proves it works end-to-end. See CONST-035 in CONSTITUTION.md.

Status legend

  • active — anchor exists and is callable; capability is verified.
  • pending-anchor — capability declared, anchor test does not yet exist. Listed in challenges/baselines/bluff-baseline.txt Section 3. Reducing this state is the work of campaign sub-project 4.
  • retired — capability removed; row kept for history.

Path format

For Go tests: <path>.go::<TestFuncName>. Verifier greps for func <TestFuncName>\b in the file.
